## Instalação 

#### Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
Git, Node, npm

#### Clone o repositório

```bash
$ git clone https://github.com/guitavano/frontendTest.git
```

#### Acesse a pasta do projeto

```bash
$ cd frontendTest
```

#### Instale os pacotes

```bash
$ npm install
```

#### Rode o projeto

```bash
$ npm start
```
